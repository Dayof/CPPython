# CPPython - Lexer/Parser

## Requirements

- Flex 2.6.4
- Bison 3.5.1
- GCC 9.3.0

## Usage

```bash
$ make
$ ./cppython tests/parser/valid_1.ppy  # for valid test
$ ./cppython tests/parser/invalid_1.ppy  # for invalid test
```
